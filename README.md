## 运行环境

面板仅支持主流 `amd64` | `arm64` 系统的最新版本，且随着系统版本的更新，面板也会逐步停止对旧版本的支持。

不支持 `Ubuntu`，因为其发版太频繁，难以维护。

低配机器建议使用 `Debian`，资源占用较 `RHEL` 系更低。其他机器建议使用 `RockyLinux` | `AlmaLinux`，维护周期更长。

不在下表中的其他 `RHEL` 系统，可自行尝试安装，但不保证能够正常运行，且不提供技术支持。

| 系统         | 版本 |
|------------|----|
| RockyLinux | 9  |
| AlmaLinux  | 9  |
| Debian     | 12 |
